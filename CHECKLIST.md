## 💻 نصائح الاستخدام

### للمطورين
```bash
# تحديث البيانات
npm run dev

# مراقبة التغييرات
npm run watch

# بناء للإنتاج
npm run build
```

### للاختبار
```bash
# الدخول إلى التطبيق
```
