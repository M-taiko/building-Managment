# دليل تثبيت نظام إدارة العمارات

## متطلبات التشغيل

- PHP >= 8.2
- Composer
- MySQL or MariaDB
- Node.js & NPM

## خطوات التثبيت

### 1. إعداد قاعدة البيانات

```sql
CREATE DATABASE building_management CHARACTER SET utf8mb4 COLLATE utf8mb4_unicode_ci;
```

### 2. تكوين ملف البيئة

انسخ ملف `.env.example` إلى `.env` وقم بتعديل إعدادات قاعدة البيانات:

```env
DB_CONNECTION=mysql
DB_HOST=127.0.0.1
DB_PORT=3306
DB_DATABASE=building_management
DB_USERNAME=root
DB_PASSWORD=
```

### 3. تثبيت المكتبات

```bash
# تثبيت مكتبات PHP
composer install

# تثبيت مكتبات JavaScript
npm install

# بناء ملفات الأصول
npm run build
```

### 4. توليد مفتاح التطبيق

```bash
php artisan key:generate
```

### 5. تشغيل Migrations

```bash
php artisan migrate
```

### 6. تشغيل Seeder (بيانات تجريبية)

```bash
php artisan db:seed --class=BuildingManagementSeeder
```

### 7. تشغيل السيرفر

```bash
php artisan serve
```

الآن يمكنك الوصول للنظام عبر: `http://localhost:8000`

## الوحدات الرئيسية

### 1. إدارة الشقق
- إضافة / تعديل / حذف الشقق
- عرض قائمة الشقق مع معلومات المالكين

### 2. إدارة الاشتراكات
- إنشاء اشتراكات شهرية للشقق
- تتبع المدفوعات (مدفوع / جزئي / غير مدفوع)
- حساب المتأخرات تلقائياً

### 3. إدارة المصروفات
- تسجيل جميع مصروفات العمارة
- تصنيف المصروفات حسب النوع
- حساب إجمالي المصروفات

### 4. طلبات الصيانة
- إنشاء طلبات صيانة من المقيمين
- متابعة حالة الطلبات (مفتوح / قيد التنفيذ / مكتمل)
- تحديد الأولوية (منخفض / متوسط / عالي)

### 5. لوحة التحكم

#### للمدير:
- إحصائيات شاملة (الوارد، المصروفات، الرصيد)
- عدد الشقق المتأخرة
- طلبات الصيانة المفتوحة
- رسم بياني للوارد والمصروفات الشهرية

#### للمقيم:
- مدفوعاتي
- متأخراتي
- سجل المدفوعات
- طلبات الصيانة الخاصة بي

- ✅ Laravel 12 (آخر إصدار)

- ✅ Bootstrap 5 RTL

## الأدوار والصلاحيات

### Super Admin
- إدارة جميع العمارات
- إدارة المستخدمين

### Building Admin
- إدارة شقق عمارته فقط
- إدارة الاشتراكات والمصروفات
- عرض ومتابعة طلبات الصيانة

### Resident
- عرض مدفوعاته ومتأخراته
- إنشاء طلبات صيانة
- عرض سجل المدفوعات

## المشاكل الشائعة

### خطأ في قاعدة البيانات
تأكد من إنشاء قاعدة البيانات وتعديل ملف `.env` بالبيانات الصحيحة.

### خطأ في Composer
قم بتشغيل:
```bash
composer update
```

### خطأ في NPM
قم بتشغيل:
```bash
npm install
npm run build
```
