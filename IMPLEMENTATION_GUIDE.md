# دليل التنفيذ الشامل - نظام إدارة العمارات

## 📋 نظرة عامة

تم تنفيذ نظام شامل لإدارة الاشتراكات الشهرية وتوزيع المصروفات في نظام إدارة العمارات. هذا الدليل يشرح جميع المميزات المضافة وكيفية استخدامها.

---

## ✨ المميزات المضافة

### 1️⃣ نظام أنواع الاشتراكات الشهرية

#### الوصف:
إدارة كاملة لأنواع الاشتراكات المختلفة (مصعد، حدائق، نظافة، قمامة، إلخ)

#### الصفحات:
- **الرابط**: `/subscription-types`
- **الصفحة الرئيسية**: عرض جميع أنواع الاشتراكات
- **إضافة**: نموذج لإضافة نوع اشتراك جديد
- **تعديل**: تعديل نوع اشتراك موجود
- **حذف**: حذف نوع اشتراك

#### الحقول:
- اسم الاشتراك (مثال: مصعد، حدائق)
- الوصف
- المبلغ الشهري
- الحالة (نشط/غير نشط)

---

### 2️⃣ صفحة تسجيل الدفعات

#### الوصف:
صفحة متخصصة لتسجيل دفع الاشتراكات الشهرية مع فلاتر متقدمة

#### الرابط: `/subscription-payments`

#### المميزات:
- ✅ فلترة حسب السنة
- ✅ فلترة حسب الشهر
- ✅ فلترة حسب نوع الاشتراك
- ✅ عرض إحصائيات (مدفوع، معلق، متأخر)
- ✅ زر تسجيل الدفع لكل اشتراك
- ✅ زر إرسال تذكير للمتأخرين

#### كيفية الاستخدام:
1. اختر السنة والشهر ونوع الاشتراك
2. اضغط "تطبيق"
3. ستظهر قائمة بجميع الاشتراكات
4. اضغط "تسجيل الدفع" لتسجيل دفع اشتراك معين
5. اضغط "إرسال تذكير للمتأخرين" لإرسال إشعارات لجميع المتأخرين

---

### 3️⃣ نظام توزيع المصروفات

#### الوصف:
نظام متطور لتوزيع المصروفات إما على جميع السكان أو على ساكن محدد

#### المميزات الجديدة:

**أ. عند إضافة مصروف:**
- اختيار نوع التوزيع:
  - **جميع السكان**: يتم تقسيم المبلغ بالتساوي
  - **شقة محددة**: يتم تحميل المبلغ كاملاً على شقة واحدة
- اختيار نوع المصروف (لمرة واحدة أو متكرر)

**ب. صفحة توزيع المصروف:**
- **الرابط**: `/expenses/{id}/shares`
- عرض تفاصيل المصروف
- جدول بتوزيع الحصص على كل شقة
- المبلغ المطلوب من كل شقة
- حالة الدفع لكل شقة
- زر تسجيل الدفع لكل حصة

#### كيفية الاستخدام:
1. من صفحة المصروفات، اضغط "إضافة مصروف"
2. املأ البيانات واختر نوع التوزيع
3. بعد الحفظ، سيتم:
   - إنشاء حصص تلقائياً لكل شقة (أو للشقة المحددة)
   - إرسال إشعارات لكل ساكن
4. اضغط "عرض الحصص" لرؤية التوزيع
5. سجل الدفع لكل حصة على حدة

---

### 4️⃣ التوليد التلقائي للاشتراكات الشهرية

#### الأمر:
```bash
php artisan subscriptions:generate-monthly
```

#### الوظيفة:
- يولد اشتراكات شهرية لجميع الشقق
- لجميع أنواع الاشتراكات النشطة
- يرسل إشعارات تلقائية لكل ساكن

#### خيارات الأمر:
```bash
# توليد للشهر والسنة الحاليين
php artisan subscriptions:generate-monthly

# توليد لشهر وسنة محددين
php artisan subscriptions:generate-monthly --month=3 --year=2026
```

#### جدولة تلقائية:
أضف إلى `app/Console/Kernel.php`:
```php
protected function schedule(Schedule $schedule)
{
    // تشغيل تلقائي في أول يوم من كل شهر
    $schedule->command('subscriptions:generate-monthly')
             ->monthlyOn(1, '00:00');
}
```

## 🗂️ هيكل قاعدة البيانات

### الجداول الجديدة:

#### 1. `subscription_types`
```sql
- id
- tenant_id
- name (اسم الاشتراك)
- description
- amount (المبلغ الشهري)
- is_active (نشط/غير نشط)
- timestamps
```

#### 2. `expense_shares`
```sql
- id
- tenant_id
- expense_id
- apartment_id
- share_amount (حصة الشقة)
- paid (تم الدفع؟)
- paid_at (تاريخ الدفع)
- timestamps
```

### الحقول المضافة:

#### جدول `subscriptions`:
- `subscription_type_id` - ربط بنوع الاشتراك
- `paid_at` - تاريخ الدفع

#### جدول `expenses`:
- `distribution_type` - نوع التوزيع (all/specific)
- `apartment_id` - الشقة المحددة (في حالة specific)
- `is_one_time` - مصروف لمرة واحدة أم متكرر

## 🔔 نظام الإشعارات

### الإشعارات التلقائية:

#### 1. عند إنشاء اشتراك شهري:
- يُرسل إشعار للساكن بنوع الاشتراك والمبلغ

#### 2. عند تسجيل دفعة:
- يُرسل إشعار تأكيد للساكن

#### 3. عند إنشاء مصروف موزع:
- يُرسل إشعار لكل ساكن بحصته من المصروف

#### 4. عند تسجيل دفع حصة مصروف:
- يُرسل إشعار تأكيد للساكن

#### 5. التذكيرات اليدوية:
- من صفحة تسجيل الدفعات
- زر "إرسال تذكير للمتأخرين"
- يرسل إشعار لجميع المتأخرين عن الدفع

## 📊 التقارير والإحصائيات

### صفحة تسجيل الدفعات:
- **مدفوع**: عدد الاشتراكات المدفوعة
- **معلق**: عدد الاشتراكات المعلقة
- **متأخر**: عدد الاشتراكات المتأخرة

### صفحة توزيع المصروف:
- **تم الدفع**: عدد الحصص المدفوعة
- **لم يتم الدفع**: عدد الحصص غير المدفوعة

## 🚀 خطوات البدء

### 1. إعداد أنواع الاشتراكات:
```
الذهاب إلى: أنواع الاشتراكات
إضافة الأنواع المطلوبة:
- مصعد: 50 ر.س شهرياً
- حدائق: 30 ر.س شهرياً
- نظافة: 40 ر.س شهرياً
- قمامة: 20 ر.س شهرياً
```

### 2. تفعيل الجدولة التلقائية:
```bash
# تأكد من تشغيل Laravel Scheduler
* * * * * cd /path-to-your-project && php artisan schedule:run >> /dev/null 2>&1
```

### 3. توليد أول دفعة اشتراكات:
```bash
php artisan subscriptions:generate-monthly
```

### 4. استخدام صفحة تسجيل الدفعات:
```
الذهاب إلى: تسجيل الدفعات
اختيار الشهر والسنة
تسجيل الدفعات
```
